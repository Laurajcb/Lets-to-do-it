# Let's to-do it

## Introduction :
In this application, we going to discover a tool whose purpose is to help us to organize and track the tasks or duties that we have to achieve.

## Functionality

The user is going to be able to type an indefinite number of tasks, and in the same way delete them, market them as completed or not; as well as accumulate prizes as the user completes the tasks.

## What makes it different?

The focus of this project is based on extrinsic motivation, a theory elaborated by researchers Deci and Ryan in 1985, especially In the first stage, *External Motivation*; which is completely determined by external factors. The person has no control over it and performs the task only on external demand and in expectation of gratification.

## How is this possible?

- The gratification in the application is going to be the badges represented through distinctive images accompanied by words or phrases that unconsciously prompt the brain to achieve gratification, the main objective for the user is to get the most as possible, and the background meaning is the achievement of tasks in an easy way.

## Colors

Also, the colors chosen in this project are a tool that provides psychological motivational value,  for example, orange: this color of stimulation and enthusiasm, is happy and energetic, is going to give us the push that our minds need in order to complete the tasks.
